# hollow

The backend of a message board project called hollow, using gin.

Just for fun.

## Dependency
- gin
- golang-jwt
- mongodb
- godotenv
- swagger

## Run Application
```sh
go run main.go
```

## Run Swagger
```sh
swag init
```
Logs will be printed in the console.
